# 简单正则匹配

动态规划  
dp[i][j]中存储的是s[:i]和p[:j]的匹配结果

举例：`s = "aab"`，`p = "a*.*"`。

 dp     | ""     |"a"     |"a*"    |"a*."   |"a*.*"
:------:|:------:|:------:|:------:|:------:|:------:
""      | T      |F       |T       |F       |T
"a"     | F      |T       |T       |T       |T
"aa"    | F      |F       |T       |T       |T
"aab"   | F      |F       |F       |T       |T
